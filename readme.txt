Embed your react app in wordpress or create single page react app.

== Description ==
- This plugin aims to ease the process of embedding react-app in wordpress page, and also create the single page react-app

Instruction on how to upload your react app:
 1- need to use npm package react-app-rewired
 2- add config-overrides.js to root directory of your react app
 3- upload the build to react post type
 4- shortcode will be generated for your to embed the react app
